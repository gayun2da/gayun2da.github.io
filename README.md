## 마크다운이란?

2004년, 존 그루버와 에런 스워츠가 만든 마크다운(markdown)은 일반 텍스트 기반의 경량 마크업 언어. 일반 마크업 언어에 비해 문법이 쉽고 간단한 것이 특징이다.

### 1. 장점 
    - 간결하다
    - 다양한 형태로 변환 가능하다


### 2. 단점
    - 표준이 없다.
    - 모든 HTML 마크업을 대신하지 못한다.

등등이 있겠습니다 :)
