## 마크다운이란?

2004년, 존 그루버와 에런 스워츠가 만든 마크다운(markdown)은 일반 텍스트 기반의 경량 마크업 언어이며, 일반 마크업 언어에 비해 문법이 쉽고 간단한 것이 특징이라고 합니다.

### 1. 장점 
    - 간결하다
    - 다양한 형태로 변환 가능하다


### 2. 단점
    - 표준이 없다.
    - 모든 HTML 마크업을 대신하지 못한다.

등등이 있겠습니다 :)


##### 이제 마크다운의 문법들에 대해 알아볼 차례입니다!

## 02 줄바꿈
줄바꿈에는 두 가지 방법이 있습니다 !

#### 1. 문장 뒤에 스페이스바 두 번

```
너무 배고프네요.  
마라탕이 먹고싶다..!
```

너무 배고프네요.  
마라탕이 먹고싶다..!

#### 2. 문장 뒤에 줄바꿈 태그인 `<br>`을 사용

```
너무 배고파요 ㅠ. ㅠ <br> 마라탕이 먹고싶다..!
```

너무 배고파요 ㅠ. ㅠ  <br>  마라탕이 먹고싶다..!

`<br>은 줄바꿈해주는 HTML 태그입니다 !`
